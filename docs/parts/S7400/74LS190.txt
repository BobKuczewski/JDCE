This device contains a 4-bit presettable up/down decade counter
with up/down mode control.  To count the /CTEN input must be low.
The counter counts up when U/D is held low and counts down when
U/D is high.  The starting value may be set by placing the starting
count on D-A and then taking /LOAD low.
